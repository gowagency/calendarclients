# Gow Calendar

Plataforma de calendário de conteúdo para clientes da Gow Agency: aprovação de posts, visão mensal e organização de entregas por cliente.

## Stack

TypeScript, Vite, React (client), Express (server) e Drizzle ORM.

## Rodando local

1. `npm install`
2. Copie `.env.example` para `.env` e preencha as variáveis
3. `npm run dev`

Produção conectada a este repositório.
